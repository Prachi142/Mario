**MARIO STYLE 2D GAME**

This project is a simple 2D Mario-style game developed using **C++ and SFML**.
The main goal of this project was to explore the basics of game development and understand how graphics, movement, and game logic work together in a real-time environment.

The game allows the player to move a character, interact with elements in the environment, and experience basic gameplay mechanics similar to classic platform games.

 **FEATURES**

* Character movement (left, right, and jump)
* Basic collision detection
* Real-time rendering using SFML
* Simple game loop structure
* Multiple objects handled using vectors

**TECH STACK**

* C++
* SFML (Simple and Fast Multimedia Library)**
* Object-Oriented Programming**
* Basic Data Structures**

 **WHAT I LEARNED**

While working on this project, I learned how to:

* Work with the SFML library for graphics rendering
* Implement a basic game loop
* Handle player movement and object interaction
* Manage multiple game objects using vectors
* Collaborate with team members on a development project

**HOW TO RUN THE PROJECT**

1. Install the SFML library on your system.
2. Clone this repository.
3. Link SFML with your IDE or compiler.
4. Build and run the project.
